Primitive Data Type
===================

  1. number --> 8 bytes--->number
  2. string  --->2 bytes per character ---> string
  3. boolean  ---> 4 bytes --> boolean
  4. null  --> 4 bytes  --->object
  5. undefined  ---> 4 bytes  --->undefined
  6. bigint   ----> bigint
  7. symbol  ---> 8 bytes
